** 表格
| 快捷键         | 说明                             |
|----------------+----------------------------------|
| C-c 坚线       | 创建或者转换成表格               |
| C-c C-c        | 调整表格，不移动光标             |
| TAB            | 移动到下一个区域，必要时新建一行 |
| S-TAB          | 移动到上一个区域                 |
| RET            | 移动到下一行，必要时新建一行     |
| M-LEFT/RIGHT   | 移动列                           |
| M-UP/DOWN      | 移动行                           |
| M-S-left/right | 删除/插入行                      |
| M-S-UP/DOWN    | 删除/插入列                      |
| C-c -          | 添加水平分割线                   |
| C-c RET        | 添加水平分割线并且跳至下一行     |

** 列表

- 无序列表项以'-', '+', '*' 开头。
- 有序列表以'1.', '1)'开头。
- 描述列表用'::'将项和描述分开.
- 有序列表和无序列表都以缩进表示层级，只要对齐缩进，不管是换行还是分块都认为是处于当前列表项。
